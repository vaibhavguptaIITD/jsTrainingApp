Control Flow

If / else

Syntax
if (/* Some condition */) {
    // Do something
} else if (/* Some other condition */) {
    // Do something else
} else {    // Otherwise
    // Do a third thing
}

Exercise

Write an if / else statement inside the isEven function. It should return true; if the number it receives is evenly divisible by 2. Otherwise (else), it should return false;.

Use % operator to check for evenness. 

6 % 2 == 0 //evaluates to true
5 % 2 == 0 //evaluates to false
