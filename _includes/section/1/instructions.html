Data types

Data comes in various types in javascript. As in any other programming language javascript comes with following datatypes:

  • Numbers

    Numbers are quantities. You can do math with them.

  • Strings

    Strings are sequences of characters, like the letters a-z , spaces, and even numbers. These are all strings: "Vaibhav" , "4" and "What is your name?" Strings are extremely useful as labels, names, and content for your programs.

  • Boolean

    A boolean can have only two values, true or false. You can use them in your code by making statements that evaluate to true or false.

  • undefined

    undefined means a variable has been declared but has not yet been assigned a value.

  • null

    null is an assignment value. It can be assigned to a variable as a representation of no value.
